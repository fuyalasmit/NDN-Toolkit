# Notebooks Overview

- `data_preprocessing.ipynb`: Cleans and prepares raw feature data before training.
- `data_analysis.ipynb`: Quick exploratory analysis of feature distributions and patterns.
- `if_model.ipynb`: Main Isolation Forest training, evaluation, multi-dataset scoring, and realtime simulation.
- `result_analysis.ipynb`: Visual analysis of generated CSV outputs with comparison charts.
- `ndn_anomaly_detection.ipynb`: Reference used for model training. Just a prototype.
